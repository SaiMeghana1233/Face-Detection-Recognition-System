# Face Detection & Recognition System

## 📌 Overview
This project detects and recognizes faces using OpenCV and Python.

## 🚀 Features
- Real-time face detection
- Face recognition using trained dataset
- Webcam integration

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python face_recognition_system.py

## 📷 Output
- Detects and labels faces in real-time
